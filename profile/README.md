# SureShotAI

**An end-to-end solution for LLM orchestration, monitoring, and metrics.**

SureShotAI is a self-hosted platform for running and observing LLM apps across
**OpenAI, Anthropic, Gemini, Groq, and Ollama** — provider orchestration plus
usage, cost, latency, model analytics, trace debugging, and error monitoring,
all organized by **account**.

## What's built

Working, data-driven pages: **Overview · Usage & Tokens · Cost & Budgets ·
Latency · Models · Trace Explorer · Errors · Accounts**. Remaining spec areas
(Prompts, RAG, Agents, Quality, Playground, Security) are stubbed nav pages — the
trace/span data model already supports them; only the views remain.
